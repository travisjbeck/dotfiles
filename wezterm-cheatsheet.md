# WezTerm Keybindings (Colemak-DH)

**Leader Key**: `Ctrl+b`  
**Directions**: `m` (←) `n` (↓) `e` (↑) `i` (→)

## Panes
- **Split**: `Leader` + `|` (horizontal) or `-` (vertical)
- **Navigate**: `Leader` + `m/n/e/i`
- **Resize**: `Leader` + `Shift+m/n/e/i`
- **Zoom**: `Leader` + `z`
- **Close**: `Leader` + `c`
- **Rotate**: `Leader` + `Space`
- **Select**: `Leader` + `0` (visual) or `q` (show IDs)

## Tabs & Workspaces
- **Rename Tab**: `Leader` + `t`
- **New Workspace**: `Leader` + `w`
- **List Workspaces**: `Leader` + `s`

## Sessions (Resurrect)
- **Save**: `Leader` + `Shift+S`
- **Load**: `Leader` + `Shift+L`
- **Delete**: `Leader` + `Shift+D`
- **Rename**: `Leader` + `Shift+$`

## Other
- **Lazygit**: `Leader` + `g`
- **Copy Mode**: `Leader` + `x`
- **Cheatsheet**: `Leader` + `i`
